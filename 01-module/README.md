## Вставка JS в Browser

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>Страница с JavaScript</title>
    <link href="css/style.css" rel="stylesheet">
</head>
<body>
    <h1>Это страница с JavaScript</h1>
    <script src="js/app.js"></script>
</body>
</html>
```
