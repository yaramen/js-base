## DOM (Document Object Model)
DOM – это представление HTML-документа ввиде дерева вложенных HTML-элементов и их содержимого
```html
<html>
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
    <div id="root">
        <button class="button">Click</button>
        <p class="text">Hello, world!</p>
        <ul>
            <li>JavaScript</li>
            <li>TypeScript</li>
        </ul>
    </div>
</html>
```
